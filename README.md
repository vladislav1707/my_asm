пример вызова ассемблера из терминала:
my_asm counter.txt counter_bin.txt bin

комментарии делаются с помощью # и то что идет после # будет игнорировано до следующей строки

команды(запятые между аргументами не нужны, просто пробелы):
label(labelName: название label):
    label labelName - позволяет сохранить место в коде и использовать в любом месте кода,
    хоть до хоть после label, очень полезно для прыжков в определенные точки кода

const(constName: название const; constValue: значение константы)
	const constName 67 - позволяет заменить все последующие упоминания константы на число в втором аргументе

логика/математика(не требует аргументов): # проводится с рег1 и рег2 и выводится в рег3
    OR   - 01000000
    NOR  - 01000001
    AND  - 01000010
    NAND - 01000011
    ADD  - 01000100
    SUB  - 01000101
    XOR  - 01000110
    SHR  - 01000111 # сдвиг вправо, игнорирует значение из рег2 в отличие от остальных

моментальная запись в рег0(аргумент: число от -32 до 31):
    IMM аргумент - 00 + число из 6 бит в двоичной системе

коопирование(откуда: строка содержащая название региста/входа откуда будет взято значение;
куда: строка содержащая название региста/выхода куда будет помещено значение):
    MOV откуда куда - 10 + число из 3 бит откуда + число из 3 бит куда

список доступных названий для MOV:
    R0        - 000
    R1        - 001
    R2        - 010
    R3        - 011
    R4        - 100
    R5        - 101
    INP1/OUT1 - 110
    INP2/OUT2 - 111

прыжок по условию(не требует аргументов):
    JMP       - 000    (always)
    JEQ/JZ    - 001    (=0)
    JNE/JNS   - 010    (!=0)
    JLT       - 011    (<0)
    JLE       - 100    (<=0)
    JGT       - 101    (>0)
    JGE       - 110    (>=0)
    NOP       - 111    (never)
